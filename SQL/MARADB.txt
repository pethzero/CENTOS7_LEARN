เมื่อคุณลืมรหัสผ่านหรือไม่สามารถเข้าสู่ระบบ MySQL/MariaDB ได้ คุณสามารถทำการเปลี่ยนหรือรีเซ็ตรหัสผ่านได้โดยตามขั้นตอนนี้:

1. หยุด MySQL/MariaDB:

สิ่งแรกที่คุณควรทำคือหยุด MySQL/MariaDB เพื่อเป็นการรักษาความปลอดภัยและให้คุณสามารถเปลี่ยนรหัสผ่านได้:

```bash
sudo systemctl stop mariadb
```

2. เริ่มเครื่อง MySQL/MariaDB ในโหมดยกเลิกการตรวจสอบความปลอดภัย (Safe Mode):

```bash
sudo mysqld_safe --skip-grant-tables &
```

คำสั่งนี้จะเริ่มเครื่อง MySQL/MariaDB ในโหมดที่ไม่ตรวจสอบความปลอดภัยและไม่จำเป็นต้องใส่รหัสผ่าน.

3. เปิดหน้าต่างคำสั่งใหม่และเข้าสู่ MySQL/MariaDB:

```bash
mysql -u root
```

4. เปลี่ยนหรือรีเซ็ตรหัสผ่านสำหรับผู้ใช้ root ด้วยคำสั่ง SQL:

```sql
USE mysql;
UPDATE user SET password=PASSWORD("new_password_here") WHERE User='root';
FLUSH PRIVILEGES;
```

แทน `new_password_here` ด้วยรหัสผ่านใหม่ที่คุณต้องการใช้.

5. ออกจาก MySQL/MariaDB:

```sql
EXIT;
```

6. หยุด MySQL/MariaDB ที่กำลังทำงานในโหมด Safe Mode:

หากคุณกำลังใช้หน้าต่างคำสั่งที่เปิดไว้ในขั้นตอนที่ 2 ให้ปิดหน้าต่างนั้น.

7. เริ่ม MySQL/MariaDB ใหม่:

```bash
sudo systemctl start mariadb
```

เมื่อทำเสร็จแล้ว คุณควรสามารถเข้าสู่ระบบ MySQL/MariaDB ด้วยรหัสผ่านใหม่ที่คุณตั้งไว้ได้แล้ว. โปรดจำไว้ว่ากระบวนการนี้เป็นกระบวนการเปลี่ยนหรือรีเซ็ตรหัสผ่านที่มีความเสี่ยง และควรใช้เพื่อความต้องการสุดท้ายเท่านั้น.
